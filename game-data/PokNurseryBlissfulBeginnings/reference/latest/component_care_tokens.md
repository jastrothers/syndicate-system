# Care Values (Dual-Use Currency)

**Care Tokens have been replaced by dual-use care cards.** Every care card now serves as both a care icon provider (when tucked under a Pokémon) and currency (when spent from hand to make purchases).

## Care Value by Tier

| Card Tier | Icons | Care Value (Currency) |
|---|---|---|
| Starting (Gentle Touch, Berry Snack, etc.) | 1 (or wild) | 1 |
| Common (Oran Berry, Potion Drops, etc.) | 1 | 1 |
| Midrange (Sitrus Berry, Full Restore, etc.) | 2 | 2 |
| Premium (Lum Berry Feast, Healing Pulse, etc.) | 3 | 3 |
| Hybrid (Pecha Berry Remedy, etc.) | 2 split + effect | 1 |

## Usage
- During Caring Hands, each card can be **tucked** (use care icons) or **spent** (use Care Value as currency), not both
- Cards spent as currency go to personal discard pile (cycle back through deck)
- To purchase, spend cards whose Care Values total >= the printed cost
- Overpayment is allowed; there is no change

## Purchasing Power Bonuses
- Berry Garden facility: Berry cards have +1 Care Value when spent as currency
- Chansey ability: Once/Day, one card spent has +1 Care Value
- Roselia ability: Once/Day, when tucking Berry, one hand card gains +1 Care Value this Day
- Blossom Greenhouse facility: If 3+ cards tucked this Day, next purchase costs 1 less
