# Tableau Building

Nursery starts with 3 slots, expandable to 5 via facility upgrades. 1 baby Pokémon per slot. Evolved Pokémon remain in slot (flip to evolved side), grant ongoing abilities and scoring. Max 3 facility cards provide bonuses.
