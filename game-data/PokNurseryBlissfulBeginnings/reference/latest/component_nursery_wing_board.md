# Nursery Wing (Player Board)

**4 boards, 280x180mm (11x7 inches)**

## Layout Zones
1. **Crib Row** (center): 5 card-sized slots horizontal. Slots 1-3 open, slots 4-5 locked (lock icon). Unlocked by Crib Expansion facility.
2. **Facility Row** (top): 3 card-sized slots for Facility Cards.
3. **Draw Bonus Reference** (bottom edge): Printed formula — Draw = 5 base + 1/evolved + facility/ability bonuses. Trailing: +1/deficit (max +3).
4. **Deck Zone** (right): Marked areas for personal draw pile and discard pile.
5. **Daily Schedule Zone** (below cribs): Open area for splaying current hand of cards.

## Scaling
- 1 per player (1-4 boards used)
- All boards are identical
