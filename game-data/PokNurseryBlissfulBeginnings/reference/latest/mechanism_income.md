# Income

Base: 2 care tokens/round. +1 per baby Pokémon in nursery. +1 per evolved Pokémon (stacks with baby bonus = +2 total). Facility bonuses vary. Token cap: 12. Creates engine-building arc where early investments compound.

**Catch-Up Bonus**: If you trail the leader in evolutions, gain +1 token per deficit (max +2). Cannot trigger on consecutive Days — place any Care Token face-down as a cooldown marker when received; remove at start of next Day.
