# Story Driven
**Type:** narrative_engine
**Mechanism ID:** story_driven
**Complexity:** 2

The game centers on a branching narrative where player choices determine the path and available resources. Narrative "Hubs" present choices that require specific crew assignments or item discards to unlock.
