# Mechanism: Hand Management
ID: hand_management
Description: Players manage a hand of 5 'Drama Cards'. Cards can be used for their action, discarded for 'Clout', or saved for the 'Reunion' phase.
Justification: Simulates the strategic timing of causing drama or playing peace-maker.
Complexity: 2
