# Archetype: The Hacker

*Bypasses digital security systems, manipulates data, and interacts with the mainframe.*

## Base Stats

- **Starting Health:** 3
- **Base Hand Size:** 5
- **Passive Skill:** *Override* - Whenever you overcome a "Digital" Obstacle, generate 1 extra Intel.

## Starting Deck (10 Cards)

### 4x Basic Script

- **Type:** Starting Card
- **Effect:** Provides 1 Intel

### 3x Burner Phone

- **Type:** Starting Card
- **Effect:** Provides 1 Gear

### 2x Social Engineering

- **Type:** Starting Card
- **Effect:** Provides 1 Intel OR reduces Heat by 1.

### 1x Overclock

- **Type:** Starting Card (Loud)
- **Effect:** Provides 2 Intel, generates 1 Heat.
