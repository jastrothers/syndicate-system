# Mechanism: Variable Player Powers
ID: asymmetric_powers
Description: Each player has a unique 'Persona' card providing a passive ability and a once-per-round 'Big Scene' power.
Justification: Reflects the distinct personalities and roles in the 'Baddies' reality show.
Complexity: 3
