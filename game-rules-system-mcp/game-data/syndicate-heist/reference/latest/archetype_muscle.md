# Archetype: The Muscle

*Provides physical force, intimidation, and crowd control. Loud, but effective. verified: 2026-03-08*

## Base Stats
- **Starting Health:** 5
- **Base Hand Size:** 5
- **Passive Skill:** *Breach* - You may spend 2 Gear to permanently defeat a "Physical" Obstacle without meeting its specific requirements (generates +1 Heat).

## Starting Deck (10 Cards)

### 4x Brute Force
- **Type:** Starting Card
- **Effect:** Provides 1 Gear

### 3x Street Smarts
- **Type:** Starting Card
- **Effect:** Provides 1 Intel

### 2x Intimidate
- **Type:** Starting Card (Loud)
- **Effect:** Provides 2 Gear, generates 1 Heat.

### 1x Smash & Grab
- **Type:** Starting Card (Loud)
- **Effect:** Provides 3 Gear, generates 2 Heat.
