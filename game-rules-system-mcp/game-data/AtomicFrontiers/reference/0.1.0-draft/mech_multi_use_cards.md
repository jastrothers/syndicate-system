# Gadget Cards (Multi-Use Cards)
**Type:** tactical_options
**Mechanism ID:** multi_use_cards
**Complexity:** 3

Each 'Gadget Card' has three uses:
1. **Immediate Effect**: Discard for a one-time resource or move.
2. **Equipment**: Attach to a Crew Member for a permanent +1 to specific checks.
3. **Narrative Shortcut**: Discard at a Narrative Hub to bypass a requirement.
