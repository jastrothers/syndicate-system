# Atomic Checks (Dice Rolling)
**Type:** resolution
**Mechanism ID:** dice_rolling
**Complexity:** 1

Encounters and challenges are resolved using a 2d6 'Atomic Check'. 
- **Success (8+)**: Fully resolve the challenge.
- **Partial (6-7)**: Resolve with a concession or minor setback.
- **Failure (5-)**: Narrative setback or ship damage.
Bonuses from Crew Assignment and Gadget Cards add to the total.
