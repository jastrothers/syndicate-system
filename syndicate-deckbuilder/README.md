# Syndicate Deckbuilder System

A strategic deckbuilding game focused on planning and executing high-stakes heists.

## Project Structure

- `PARKING_LOT.md`: Brainstorming and early-stage ideas.
- `planning/`: Project roadmaps and progress management.
  - [implementation_plan.md](./planning/implementation_plan.md): Detailed milestones and technical plan.
  - [progress_management.md](./planning/progress_management.md): Long-term tracking and synchronization.
- `rules/`: Informal and formal rulebook sections.
  - [CoreRules.md](./rules/CoreRules.md): The theme-agnostic core gameplay rules.
  - [card-schema.json](./rules/card-schema.json): The formal data structure defining all card types in the game.
- `cards/`: (Planned) Card definitions and references.

## Getting Started

Begin by reviewing the [CoreRules.md](./rules/CoreRules.md) for the foundational theme-agnostic mechanics. For ongoing brainstorming, thematic mappings, and future ideas, check [PARKING_LOT.md](./PARKING_LOT.md).
