# Crew Assignment (Worker Placement)
**Type:** action_selection
**Mechanism ID:** worker_placement
**Complexity:** 3

Players assign 'Crew Tokens' (Captain, Science Officer, Cadet) to 'Duty Stations' on the ship or planetary Map.
- **Captain**: Bonus to Diplomacy/Lead checks.
- **Science Officer**: Bonus to Tech/Nature checks.
- **Cadet**: Flexible but weaker bonuses.
Stations determine which Narrative Hub choices are accessible.
