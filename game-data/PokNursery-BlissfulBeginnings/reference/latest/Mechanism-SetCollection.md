# Set Collection

**Taxonomy:** Cards and Deck Management > set_collection | Complexity: 2

## Parameters

- **Evolution Recipes:** Each baby Pokemon card specifies a combination of Care Tokens needed to evolve.
  - Example: Pichu = 3 Joy + 2 Food + 1 Rest (6 tokens) -> Pikachu
  - Example: Togepi = 2 Joy + 2 Food + 2 Rest (6 tokens) -> Togetic
  - Example: Happiny = 1 Joy + 1 Food + 4 Rest (6 tokens) -> Chansey
- **Recipe Range:** 5 to 7 total tokens per baby, with distributions reflecting personality.
- **Harmony Bonus:** Evolving Pokemon of the same type family grants +2 VP per matched pair.
- **Supply Card Conversion:** Supply cards substitute for any token type at a 2:1 ratio.

## Justification

Set collection is the heart of the nurturing loop. Each baby Pokemon is a puzzle solved through care. Varying recipes encourage specialization or diversification. The Harmony Bonus adds a secondary collection layer rewarding thematic coherence. Complexity stays low because recipes are printed directly on cards.

## Synergies

- **Action Points:** AP spent on care actions directly fill collection requirements.
- **Tableau Building:** Completed sets trigger evolution, populating the Evolution Garden.
- **Drafting:** Choosing which babies to adopt is choosing which sets to pursue.
