# Tableau Building

**Taxonomy:** Economy and Resources > tableau_building | Complexity: 3

## Parameters

- **Nursery Board** (personal tableau per player):
  - **4 Cradle Slots** for baby Pokemon (expandable to 5 via Poke Ball Storage upgrade).
  - **3 Upgrade Slots** for nursery improvements.
  - **1 Manager Slot** for Chansey/Blissey/Wigglytuff/Audino card.
  - **Evolution Garden** where evolved Pokemon are placed (scoring zone).
- **Nursery Upgrades** (8 available, drafted every 3rd round):
  - *Berry Garden*: +1 free Food token at start of each turn.
  - *Playroom*: +1 free Joy token at start of each turn.
  - *Soft Beds*: +1 free Rest token at start of each turn.
  - *Poke Ball Storage*: Increase cradle capacity to 5.
  - *Day Care Center*: Once per turn, move 1 token between babies.
  - *Healing Bell*: On evolution, gain 1 token of choice on another baby.
  - *Lucky Egg*: Evolved Pokemon score +1 VP each.
  - *Friendship Bracelet*: Harmony Bonuses increase to +3 VP.

## Justification

The nursery board IS the game's spatial metaphor. Players see their nursery -- babies in cradles with tokens accumulating, upgrades improving efficiency. Creates a satisfying engine-growth arc: early game is adopting/gathering, mid game is optimizing with upgrades, late game is pushing evolutions. Complexity managed by fixed slots.

## Synergies

- **Set Collection:** Completed sets trigger evolution, populating the Evolution Garden.
- **Drafting:** Meadow draft feeds cradle slots; Upgrade draft feeds upgrade slots.
- **Income:** Upgrade tiles augment passive income generation.
