# Archetype: The Fixer

*The logistical mastermind. Excels at resource management and specialized gear procurement.*

## Base Stats

- **Starting Health:** 3
- **Base Hand Size:** 5
- **Passive Skill:** *Connections* - Cards cost 1 less of either currency type in the Black Market.

## Starting Deck (10 Cards)

### 4x Supplies

- **Type:** Starting Card
- **Effect:** Provides 1 Gear

### 3x Blueprints

- **Type:** Starting Card
- **Effect:** Provides 1 Intel

### 2x Black Market Dealer

- **Type:** Starting Card
- **Effect:** Draw 1 extra card next turn during The Legwork, OR provides 1 Gear.

### 1x Safehouse

- **Type:** Starting Card (Quiet)
- **Effect:** Reduce your Heat by 2. Limit: Evaluate once per Execution phase.
